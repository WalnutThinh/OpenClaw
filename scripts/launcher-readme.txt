EClaw Launcher - Quick Install Guide
====================================

Files in this ZIP:
- EClaw-Launcher.exe
- README-INSTALL.txt

Install steps:
1) Extract this ZIP to a normal folder (for example: D:\EClaw-Launcher\).
2) Run EClaw-Launcher.exe.
3) Choose install folder and continue.
4) Wait for download + verify + extract to finish.
5) Launch EClaw after install completes.

If Windows SmartScreen appears:
1) Click "More info".
2) Click "Run anyway".

Important:
- Download this ZIP only from the official Enchante website.
- Do not run directly from inside the ZIP preview.
- Keep latest.json and app package hosted on the official download URL.

Version notes:
- Product: EClaw
- Launcher file: EClaw-Launcher.exe
- Installer alias: EClaw-Setup.exe
